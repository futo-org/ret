How to self-host ret

You are encouraged to self host Ret and use it from your own web server.

  1. 1. Download the ret.zip release. You can find it here
  2. 2. Extract the contents into a web server.
  3. 3. That's it!

Nginx, Apache, Github Pages, Cloudflare pages, or neocities will work. The web server doesn't need any special rules, since the Ret release zip is all static content.
