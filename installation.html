How to self-host Ret

I encourage to self-host Ret on your own web server, and hack on it to make it more useful for yourself.

To self-host Ret, simply download the latest ret.zip and extract it to your web server. Ret works entirely client-side, and requires no special CORS policies (other than cross origin requests to load the wasm binary). No matter where you host Ret (Nginx, Apache, Github Pages, Cloudflare Pages, Neocities, etc) it should work.

Nightly release builds can be found here.
